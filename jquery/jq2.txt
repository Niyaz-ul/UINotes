JQuery slideDown()
---------------------
JQuery slideDown() method is used to slide down an element.
ex:
<!DOCTYPE html>
<html>
  <head>
    <title>MyPage</title>
   
    <!-- add jquery cdn -->
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js">
    </script>
    <style>
        #panel,#flip
        {
          text-align: center;
          background-color: #00ffff;
          border:1px solid #c3c3c3;
        }
        #panel
        {
          padding:50px;
          display:none;
        }
    </style>
  </head>
  <body>
          
   <div id="flip">Click to slideDown</div>

   <div id="panel">
    Hello www.globaledx.com! It is one of the best place to learn JQuery and other UI technologies to build IT career.
   </div>

   <script>
      $(document).ready(function(){
        $("#flip").click(function(){
            $("#panel").slideDown("slow");
        })
      })
   </script>

  </body>
</html>

JQuery delay()
----------------
The JQuery delay() method is used to delay the execution of functions in the 
queue.It is a best method to make a delay between the queued Jquery effects.

The delay() method is used to set the timmer to delay the execution of the 
next item in the queue.

ex:
<!DOCTYPE html>
<html>
  <head>
    <title>MyPage</title>
   
    <!-- add jquery cdn -->
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js">
    </script>

  </head>
  <body>
         <button>Click Me </button> <br>
         <div id="div1" style="width:90px;height:90px;display:none;background-color: black"></div>

         <script>
          $(document).ready(function(){
            $("button").click(function(){
              //$("#div1").delay("fast").fadeIn();
              //$("#div1").delay("slow").fadeIn();
              //$("#div1").delay(1000).fadeIn();
              $("#div1").delay(4000).fadeIn();
            })
          })
         </script>
  </body>
</html>

Jquery html()
-----------------
JQuery html() method is used to change the entire content of the selected 
elements.It replaces the selected element content with new content.

ex:
<!DOCTYPE html>
<html>
  <head>
    <title>MyPage</title>
   
    <!-- add jquery cdn -->
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js">
    </script>

  </head>
  <body>
         <button>Click Paragraph</button> <br>

         <p>This is Paragraph 1.</p>
        
          <p>This is Paragraph 1.</p> 

          <script>
            $(document).ready(function(){
               $("button").click(function(){
                $("p").html("Hello <b>GlobalEdx</b>");
               })
            })
          </script>
  </body>
</html>

JQuery val()
---------------
There are two usage of value() method.

1)It is used to set the value of every matched element.
ex:
<!DOCTYPE html>
<html>
  <head>
    <title>MyPage</title>
   
    <!-- add jquery cdn -->
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js">
    </script>

  </head>
  <body>
        <p>Name: <input type="text" name="user"/></p>
        <button>set the value</button>         
        
        <script>
            $(document).ready(function(){
              $("button").click(function(){
                $("input:text").val("GlobalEdx");
              })
            })
        </script>
  </body>
</html>

2)It is used to get the current value of the first element in the set of matched
elements.

ex:
<!DOCTYPE html>
<html>
  <head>
    <title>MyPage</title>
   
    <!-- add jquery cdn -->
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js">
    </script>

  </head>
  <body>
       <p></p>
       <select id="single">
          <option>One</option>
          <option>Two</option>
          <option>Three</option>
       </select>

       <script>
        function f1(){
          var singleValue =$("#single").val();
          $("p").html("<b>Value :</b>"+singleValue);
        }
        $("select").change(f1);
        f1();
       </script>

  </body>
</html>

Jquery before()
---------------
The jQuery before() method is used to insert the specified content before 
the selected elements. It adds the content specified by the parameter,
 before each element in the set of matched elements.
ex:
<!DOCTYPE html>
<html>
  <head>
    <title>MyPage</title>
   
    <!-- add jquery cdn -->
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js">
    </script>

  </head>
  <body>
      
      <button>add content before </button>

      <p>This is paragraph1</p>
      <p>This is paragraph2</p>


       <script>
        $(document).ready(function(){
          $("button").click(function(){
            $("p").before("<p><b>GlobalEdx</b></p>");
          })
        })
       </script> 
  </body>

</html>
