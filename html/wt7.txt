<label> element/tag
-------------------
The <label> tag defines label for many form elements.

The <label> element also help the user who have difficulty clicking on very small regions.

ex:
<!DOCTYPE html>
<html>
        <head>
               
        </head>
        <body>
               
                <form name="myform">
		<label>Name:</label>
		<input type="text" name="t1"/>
		</form>
        </body>
</html>

Form action attribute
======================
The action attribute defines the action to be performed when the form is 
submitted.

Usually, the form data of a web page sends to a server when user click the submit
button.

ex:
globaledx.html
----------------
<!DOCTYPE html>
<html>
        <head>
               
        </head>
        <body>
               
                <form name="myform" action="a.html">
		<label>UserName:</label><input type="text" name="t1"/>
		<br>
		<label>Password:</label><input type="password" name="t2"/>
		<br>
		<input type="submit" value="submit"/>
		</form>
        </body>
</html>

a.html
------
<!DOCTYPE html>
<html>
	<head>
		<title>A page</title>
	
	</head>
	<body bgcolor="purple">
		This is A page
	</body>
</html>

Form target Attribute
-------------------------
The target attribute specifies if the submitted result will open in a new 
window or tab or in the current window.


The default value is "_self" which means the form will be submitted in the 
current window.
ex:
<!DOCTYPE html>
<html>
        <head>
               
        </head>
        <body>
               
                <form name="myform" action="a.html" target="_self">
		<label>UserName:</label><input type="text" name="t1"/>
		<br>
		<label>Password:</label><input type="password" name="t2"/>
		<br>
		<input type="submit" value="submit"/>
		</form>
        </body>
</html>

To make the form result open in a new window or browser we need to use "_blank" value.

ex:
<!DOCTYPE html>
<html>
        <head>
               
        </head>
        <body>
               
                <form name="myform" action="a.html" target="_blank">
		<label>UserName:</label><input type="text" name="t1"/>
		<br>
		<label>Password:</label><input type="password" name="t2"/>
		<br>
		<input type="submit" value="submit"/>
		</form>
        </body>
</html>


form method attribute
--------------------------

ex:

		<br>
		<input type="submit" value="submit"/>
		</form>

</html>

GET Methodology
------------------
It appends the form data into the url in name/value pair.

It will store limited amount of data.

we never sent sensitive data using GET Methodology.

GET is a better for non-secure data.

Fileuploading and encryption can't be done using GET methodology.

It sends the request fastly.

POST methodology
-----------------
It does not appends the form data into the url in name/value pair.

It will store unlimited amount of data.

We can sent sensitive and non-sensitive data using POST methodology.

POST is better for secure data.

Fileuploading and encryption can be done using POST methodology.

It send the request bit slow.

HTML Textarea
--------------

</html>

Html select box
----------------
<select> tag is used to create a select box in a web page.

<option> tag is used to declare the list of options in select box.

ex:
<!DOCTYPE html>
<html>
        <head>
               
        </head>
        <body>
               
             	Courses:
		<select name="s1">
			<option>HTML</option>
			<option>CSS</option>
			<option>JS</option>
			<option>ANGULAR</option>
			
		</select>
		<br>
		Courses:
		<select name="s2">
			<option value="html">HTML</option>
			<option value="css">CSS</option>
			<option value="js">JS</option>
			<option value="angular">ANGULAR</option>
			
		</select>
        </body>
</html>
