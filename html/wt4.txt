viewport
--------
	Html5 introduced a method to let web designers take control over
	the viewport through meta tag.

	It makes our website as responsive.

syntax
-----
	<meta name="viewport" content="width=device-width,initial-scale=1.0">

ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage</title>
		<meta name="viewport" content="width=device-width,initial-scale=1.0">
	
	</head>
	<body>
		This is GlobalEdx
	</body>
</html>
Note:
-----
	Too see the output for viewport we need to enable developer option 
	by pressing F12 function key.

Basic HTML Tags
================
heading tags
---------------
	When we place the text within heading tags <h1>........</h1>.
	It will displayed on the browser window in bold format and size of 
	a text depends on the number of heading tags.

	There are six different html heading tags which are defined with 
	the <h1> to <h6> from higher level to the least level.

ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage</title>
	
	</head>
	<body>
		<h1>This is GlobalEdx</h1>
		<h2>This is GlobalEdx</h2>
		<h3>This is GlobalEdx</h3>
		<h4>This is GlobalEdx</h4>
		<h5>This is GlobalEdx</h5>
		<h6>This is GlobalEdx</h6>
	</body>
</html>

bold tag
-------
<b> tag is used to display the text in bold format.

italic tag
----------
<i> tag is used to dispaly the text in italic format.

center tag
----------
<center> tag is used to display the text in center.

underline tag
-----------
<u> tag is used to display the text in underline.

strong tag
-----------
<strong> tag is used to display the text in strong.

break line tag
---------------
<br> tag is used to break the line in a web page.

<marquee> tag
--------------
It is used to scroll to the text on a web page.

Nested Tags in HTML
-----------------------
A Tag inside another tag is called nested tag.

ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage</title>
	
	</head>
	<body>
		<center>
			<h1>
				<i>
				     <font color="red">
						Welcome To GlobalEdx
				     </font>
				</i>
			</h1>
		</center>
	</body>
</html>

Note:
-----
	In html a tag contains attributes and each attribute should be 
	seperated with space.
