Q)What is the difference between undefined and null?
ans)
A variable is said be a undefined if it has been declared but no value has
been given to it.

ex:
<!DOCTYPE html>
<html>
	<head>
			<title>MyPage!</title>
	</head>
	<body>
				<script>	
				 //callie function
				 function f1()
				 {
					var i;
					document.writeln(i);
				 }
				
				 //caller function
				 f1();
				</script>
	</body>
</html>

A variable is said be a null if we declare value as null.

<!DOCTYPE html>
<html>
	<head>
			<title>MyPage!</title>
	</head>
	<body>
				<script>	
				 //callie function
				 function f1()
				 {
					var i=null;
					document.writeln(i);
				 }
				
				 //caller function
				 f1();
				</script>
	</body>
</html>
Note:
------
Therefore "undefined" is a variable type whereas "null" is an object value.



JavaScript Operators
====================
Operators are the symbols which is used to perform some operations on operands.
ex:
	C=A+B;

	Here A,B,C are operands. 
             =,+ are operators.

There are following types of Operators in JavaScript.

1)Arithmetic Operators

2)Comparision Operators/Relational Operators

3)Bitwise Operators

4)Logical Operators

5)Assignment Operators

6)Special Operators

1)Arithmetic Operators
========================
Operator	Description				
--------	----------				
+		Addition
-		Subtraction
*		Multiplication
/		Division
%		Modulus(reminder)
++		Increment
--		Decrement

ex:
<!DOCTYPE html>
<html>
	<head>
			<title>MyPage!</title>
	</head>
	<body>
				<script>	
			
					var a=10;
					var b=5;
					document.writeln(a+b+"<br>");//10+5
					document.writeln(a-b+"<br>");//10-5
					document.writeln(a*b+"<br>");//10*5
					document.writeln(a/b+"<br>");//10/5
					document.writeln(a%b+"<br>");//10%5
				
				</script>
	</body>
</html>
ex:2
----
<!DOCTYPE html>
<html>
	<head>
			<title>MyPage!</title>
	</head>
	<body>
				<script>	
			
					var a=10;
					a++; // +1
					document.writeln(a);//11
					
					a--; // -1
					document.writeln(a);//10
				
				</script>
	</body>
</html>

>>			Bitwise Right shift


& -Truth Table
--------------
T	T	= T

T	F	= F

F	T	= F

F	F 	= F

ex:1
-----
<!DOCTYPE html>
<html>
	<head>
			<title>MyPage!</title>
	</head>
	<body>
				<script>	
				document.write((5 & 10)+"<br>");//
				/* 
					5  :  0101
					10 :  1010 <-
					-----------
					&  :  0000
							  <-
					0*1  +  0*2   +  0*4   +  0*8	
							
					0 + 0 + 0 + 0 
					= 0
				*/			
				</script>
	</body>
</html>

ex:2
----

					-----------
					&  :  1010
							  <-
					0*1  +  1*2   +  0*4   +  1*8	
							
					0 + 2 + 0 + 8 
					= 10
				*/			
				</script>
	</body>

Q)What is the difference between "==" and "===" equals operator?

ans)
==
---
	It is used to compare two variable values are same or not.

===
----
	It is used to compare two variable values as well as it will check
	type of a value are same or not.

ex:1
----
<!DOCTYPE html>
<html>
	<head>
			<title>MyPage!</title>
	</head>
	<body>
				<script>	
				
				document.writeln((0==false)+"<br>"); // true
				document.writeln((0===false)+"<br>"); // false
				/* == values should match and datatypes */
					 
				
				</script>
	</body>
</html>

ex:2
-----
<!DOCTYPE html>
<html>
	<head>
			<title>MyPage!</title>
	</head>
	<body>
				<script>	
				
				document.writeln((10=="10")+"<br>"); // true
				document.writeln((10==="10")+"<br>"); // false
					 
				
				</script>
	</body>
</html>
