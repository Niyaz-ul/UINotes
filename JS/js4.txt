JavaScript IF-Else
===================
The JavaScript If-Else statement is used to execute the code either our 
condition is true or false.

There are three forms of if statement in JavaScript.

1)If statement

2)If Else statement

3)If else IF statement

1)If statement
----------------
It evaluates/Execute the content only if expression or condition is true.

For false condition it does not execute the code/content.

syntax
------
if(expression/condition)
{
	//content to be evaluated
}

2)If Else statement
----------------------
It evaluates/execute the content/code either our condition is true or false.

syntax
------

{
//code to be evaluated for true condition
}
else
{
//code to be evaluated for false condition
}

3)If else if Statement
----------------------
It evaluates the content/code only  if expression is true from several conditions.

syntax
-----
if(expression)
{
	//content to execute
}
else if(expression)
{
	//content to execute
}
else if(expression)
{
	//content to execute
}
else
{
	//content to execute if all conditions are false	
}
ex:
<!DOCTYPE html>
<html>
	<head>
			<title>MyPage!</title>
	</head>
	<body>
				<script>	
				var a=108;
				if(a==100)
				{
					document.writeln("It is police number ");
				}
				else if(a==103)
				{
					document.writeln("It is Enquiry number ");
				}
				else if(a==108)
				{
					document.writeln("It is Emergency number");
				}
				else
				{
					document.writeln("Invalid option ");
				}
				
				</script>
	</body>
</html>

Javascript switch
====================
The javascript switch statement is used to execute the content/code from multiple
conditions.It is similar to if else if statement.

But it is convenient then if else if statement because it can be used with 
numbers and characters.

syntax
-----
switch(expression)
{
	case value1:  
			code to be execute
		        break stmt;

	case value2:
			code to be execute
			break stmt;
	-
	-
	default:
			code to be execute if all cases are false.
}


ex:

<!DOCTYPE html>
<html>
	<head>
			<title>MyPage!</title>
	</head>
	<body>
				<script>	
				var n=108;
				switch(n)
				{
					case 100: 
							  document.writeln("It is police number");
							  break;
					case 103: 
							  document.writeln("It is Enquiry number");
							  break;
					case 108: 
							  document.writeln("It is Emergency number");
							  break;
							  
					default:  document.writeln("Invalid option");		
				}
				</script>
	</body>
</html>

JavaScript LOOPS
==================
The Javascript LOOPS are used to iterate the piece of code using for,while,
do while , for-in loop.

It is mostly used in arrays.

There are four types of LOOPS in JavaScript.

1)for LOOP
2)while loop
3)do while loop
4)for-in loop

1)for LOOP
--------
It is used to execute/Iterate the code/content untill our condition is true.
syntax
----
for(initialization;condition;increment/decrement)
{
	//content to evaluate
}

ex:
<!DOCTYPE html>
<html>
	<head>
			<title>MyPage!</title>
	</head>
	<body>
				<script>	
				//array with elements
				var arr=[1,56,2,7,9];
				
				var i;
				for(i=0;i<arr.length;i++)
				{
					document.writeln("<br>"+arr[i]);
				}
				
				</script>
	</body>
</html>

2)while loop
----------
It is used to execute the code untill our condition is true.

3)do while loop
----------------
It will execute the code/content untill our condition is true.

In do while our code will execute atleast for one time either our condition is
true or false.

syntax
-----
do
{
	//code to evaluate
}while(condition/expression);

4)for-in loop
--------------
The for/In statement loops the values or properties of an object.

ex:
<!DOCTYPE html>
<html>
	<head>
			<title>MyPage!</title>
	</head>
	<body>
				<script>
					//array with elements
					//array is known as object.
					var arr=[10,20,30,40];
					var i;
					for(i in arr)
					{
						document.writeln(arr[i]+"<br>");
					}
					
				</script>
	</body>
</html>
