getElementsByName()
--------------------
It returns all the elements having the given name value.
ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage</title>
	</head>
	<body>
	
			First Name: <input name="fname" type="text" value="Alan"><br>
			<button onclick="myFunction()">Click</button>

			<p id="demo"></p>

			<script>
				function myFunction() {
 
					var x = document.getElementsByName("fname")[0].tagName;
					document.getElementById("demo").innerHTML = x;
				}
			</script>
				
	</body>
</html>

getElementsByClassName()	
-------------------------
It returns all the elements having the given class name.


ex:
<!DOCTYPE html>

				function myFunction() {
					var x = document.getElementsByClassName("test");
					x[0].innerHTML = "After Changing the Text";
				}
</script>

DOM addEventListener() Method
--------------------------------
The document.addEventListener() method attaches an event handler to the 
document.

ex:

<html>
	<head>
		<title>MyPage</title>
	</head>

					<script>
						document.addEventListener("click", function(){
						document.getElementById("result").innerHTML = "GlobalEdx";
							});
					</script>
			
	
	</body>
</html>

DOM removeEventListener() Method
------------------------------------

The document.removeEventListener() method removes an event handler that has
 been attached with the document.addEventListener() method.

ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage</title>
	</head>
	<body> 
			<h3>Click this button to stop mouseover effect !!</h3> 
   
			<button id="clickIt" onclick="RespondClick()">Click here</button> 
  
			<h1 id="hoverPara">Mouse Over text shown below !</h1> 
  
			<b id="effect"></b> 
  
    <script> 
        const y = document.getElementById("hoverPara"); 
  
        y.addEventListener("mouseover", RespondMouseOver); 
  
        function RespondMouseOver() { 
            document.getElementById("effect").innerHTML += 
                "mouseover Event !!" + "<br>"; 
        } 
  
        function RespondClick() { 
            y.removeEventListener("mouseover", RespondMouseOver); 
            document.getElementById("effect").innerHTML += 
                "Now mouseover event stoped !!" + "<br>" ; 
        } 
    </script> 
</body> 
</html>
