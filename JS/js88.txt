Methods of document object
==============================

1)write("String")
--------------------
writes the given string on the doucment.

ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage</title>
	
	</head>
	<body>
		<script>
			document.write("Hello World");
		</script>
	</body>
</html>

2)writeln("string")
--------------------
writes the given string on the doucment with space the end.
ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage</title>
	
	</head>
	<body>
		<script>
			document.writeln("Hello World");
			document.writeln("GlobalEdx");
		</script>
	</body>
</html>

Reading Element Value 
-----------------------
We can read any Element value in javascript by using "name" attribute.

ex:
    <script type="text/javascript">  
   	 function printvalue(){  
    		var name=document.form1.name.value;  
    		alert("Welcome: "+name);  
    }  
    </script>  
      
    <form name="form1">  
    	Enter Name:<input type="text" name="name"/>  
    	<input type="button" onclick="printvalue()" value="print name"/>  
    </form>  

document.getElementById() method
-------------------------------------------
It returns the element having the given id value.
ex:

    <script type="text/javascript">  
    function getcube(){  
    	var number=document.getElementById("number").value;  
    	alert(number*number*number);  
    }  
    </script>  
    <form>  
    	Enter No:<input type="text" id="number" name="number"/><br/>  
    	<input type="button" value="cube" onclick="getcube()"/>  
    </form>


How to calculate two text fields values in javascript
=============================================

<!DOCTYPE html>
<html>
	<head>		
	</head>
	<body>
		<h1>Calculate Two fields</h1>
		
		<form>
		
			No1:<input type="text" id="t1"/><br>
			No2:<input type="text" id="t2"/><br>
			<input type="button" value="calculate" onclick="calcu()"/>	
			
			<h1 id="answer"></h1>
		</form>

		<script>
			function calcu()
			{
				var val1=document.getElementById("t1").value;
				var val2=document.getElementById("t2").value;
				var result=parseFloat(val1)+parseFloat(val2);
			
				if(!isNaN(result))
				{
			document.getElementById("answer").innerHTML="The Result is :"+result;
				}
			}   
    		</script>  
		
	</body>
	
</html>

JavaScript Hide and Show a portion of a form
========================================

					function hide()
					{
							if(document.getElementById('box').checked)
							{
						document.getElementById('shopinfo').style.display='none';
							}
							else
							{
						document.getElementById('shopinfo').style.display='block';
							}
					}
