Bootstrap Alerts
==================
Bootstrap provides an easy way to create predefined alert message.

We need to create a <div> tag/element with "alert" class and one of the 
following contextual state class.

1)alert-danger

2)alert-success

3)alert-info

4)alert-warning

ex:
<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
      <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.16.0/umd/popper.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>
  </head>       
  <body>
      <div class="container">
          <div class="alert alert-danger">
            <strong>Error!</strong>There is a problem in loading a page.
          </div>

           <div class="alert alert-success">
            <strong>Success!</strong>There is a problem in loading a page.
          </div>

           <div class="alert alert-info">
            <strong>Info!</strong>There is a problem in loading a page.
          </div>

           <div class="alert alert-warning">
            <strong>Warning!</strong>There is a problem in loading a page.
          </div>

      </div> 
        
  </body>
</html>

Closing the bootstrap alerts
------------------------------
A "close" class is used to adjust the cross icon to right side.

A '&times;' it displays cross icon(X).

data-dismiss attribute is used to close the alert.

ex:
<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
      <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.16.0/umd/popper.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>


  </head>       
  <body>
      <div class="container">
          <div class="alert alert-danger">
            <a href="" class="close" data-dismiss="alert">&times;</a>
            <strong>Error!</strong>There is a problem in loading a page.
          </div>

           

      </div> 
        
  </body>
</html>

Bootstrap Buttons
=================
Bootstrap provides different styles of buttons.

To achieve the button style we need to use following classes.

1)btn

2)btn-default

3)btn-primary

4)btn-success

5)btn-info

6)btn-warning

7)btn-danger

8)btn-link

ex:
<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
      <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.16.0/umd/popper.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>

  </head>       
  <body>
      <div class="container">
         <!-- html buttons -->
         <button>html button</button>
         <input type="button" value="html btn2"/>
         <input type="submit" value="submit"/>
          <br>
        <!-- css buttons -->
        <button style="padding:10px;background-color: red">css btn1</button>    
        <br>
        <!-- bootstrap buttons -->
        <button class="btn">BT btn1</button>  
        <button class="btn-default">BT btn1</button>  
        <button class="btn-primary">BT btn1</button>
        <button class="btn-warning">BT btn1</button>        
        <button class="btn-info">BT btn1</button>   
        <button class="btn-danger">BT btn1</button> 
	<button class="btn-success">BT btn2</button>
        <button class="btn-link">BT btn1</button> 
      </div> 
        
  </body>
</html>

Bootstrap button groups
===========================
Bootstrap allows us to create a series of buttons together in a button group.

Use a <div> tag/element with class "btn-group" to create button group.

ex:

<html>

              <button class="btn btn-primary">ABOUT</button>
              <button class="btn btn-primary">SERVICES</button>

      </div>

</html>

Bootstrap Drop Down Menu
===========================
To create a bootstrap dropdown menu we need to use "btn-group" class.

A "caret" class is used to display downarrow icon.


A "data-toggle" attribute is used to display the data to drop down list.

A "dropdown-menu" class is used to adjust the drop down list items.

ex:
<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
     <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js"></script>

  </head>       
  <body>
    <br>
    <br>
      <div class="container">
        <button class="btn btn-primary">HOME</button>  
         <button class="btn btn-primary">ABOUT</button> 

         <div class="btn-group">
          
          <button class="btn btn-primary" data-toggle="dropdown">
            Services<span class="caret"></span>
          </button> 
          <ul class="dropdown-menu">
                <li><a href="#">Link1</a></li>
                <li><a href="#">Link2</a></li>
                <li><a href="#">Link3</a></li>
          </ul>

          </div>


      </div> 
        
  </body>
</html>

Bootstrap header with drop down menu
---------------------------------------
We can take headers in dropdown menu.

To take header in dropdown menu we need to use "dropdown-header" class.

Headers are not selected in drop down menu.

ex:

<!DOCTYPE html>
<html>

      <title>MyPage!</title>

  </head>

    <br>
