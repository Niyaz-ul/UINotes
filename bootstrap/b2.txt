Bootstrap sizing
================
IT is used to size the element by using sizing utility classes.

we can easily make an element wide or a toll with our width and height utility.

It suports for 25%,50%,75% and 100% bydefault.

ex:
<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
      <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.16.0/umd/popper.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>

  </head>       
  <body>
      <!-- width -->
      <div class="w-25 bg-dark text-white">width for 25%</div>
       <div class="w-50 bg-dark text-white">width for 50%</div>
        <div class="w-75 bg-dark text-white">width for 75%</div>
         <div class="w-100 bg-dark text-white">width for 100%</div>
  </body>
</html>

Bootstrap spacing
====================
Spacing utility is used to assign the responsive friendly margin and padding
values to an element or a subset of its sids with shorthands classes.

In include individual properties ,all properties , vertical properties and 
horizontal properties.

Bootstrap padding
--------------------
p- It is used for classes that set padding.

"sides " is one of the following.

t- It is used to set the padding top.

b- It is used to set the padding bottom.

l- It is used to set the padding left.

r- It is used to set the padding right.

x- It is used to set the padding left and right.

y- It is used to set the padding top  and  bottom.

blank- It is used to set the padding all 4 sides.


"Size" is one of the following.

0 - It is used to set the padding at 0.

1 - It is used to set the padding at .25.

2 - It is used to set the padding at .5.

3 - It is used to set the padding at $spacer x and $spacer y.

4 - IT is used to set the padding at $spacer x 1.5 and $spacer y 1.5.

5 - IT is used to set the padding at $spacer X 3 and $spacer y 3.


ex:
<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>

  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>

  </head>       
  <body>
        <div class="p-0  bg-primary text-white">bg primary</div>
         <div class="p-1 bg-secondary text-white">bg secondary</div>
          <div class="p-2 bg-primary text-white">bg primary</div>

           <div class="p-4 bg-primary text-white">bg primary</div>
            <div class="p-5  bg-secondary text-white">bg secondary</div>

  </body>
</html>

Bootstrap container
====================
Containers are used to pad the content inside of them.

In bootstrap there are two types of container classes.

1)container-fluid

2)container

1)container-fluid
-------------------
The "container-fluid" class provides A full with container,extends the entire
width of the viewport.

ex:
<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
      <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.16.0/umd/popper.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>

  </head>       
  <body>
      <!-- bootstrap container fluid -->
      <div class="container-fluid bg-primary">
            <h1>This is GlobalEdx</h1>
            <p>This is a bootstrap container class</p>

  </body>
</html>

Bootstrap 4 Grid System
==========================
Bootstrap's grid system is built with flexboxes.

To create a row in a grid system we need to use "row" class.

It allows us to create 12 columns in a single row and it is not required that 
we must use all 12 columns.

Diagram: gridsystem

The grid system is responsive and column will arrange automatically dependent
upon the screen size.

Grid class
-----------
According to Bootstrap4 , A grid  system contains 5 classes.

col  - extra small device.


col-sm - small devices

col-md - medium devices.

col-lg - large devices

col-xl - extra large devics.

ex: 
-----

<html>
