Bootstrap nav component
=======================
Bootstrap nav component makes it very easy to create a navigation 
components i.e navigation tabs or navigations pills.

ex:
In html , a menu is often defined in an unordered list <ul> tag.
ex:
<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
     <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js"></script>
      
  </head>       
  <body >
    <div class="container">
          <ul>
              <li><a href="#">Menu1</a></li>
              <li><a href="#">Menu2</a></li>
              <li><a href="#">Menu3</a></li>
              <li><a href="#">Menu4</a></li>
          </ul>
    </div>
  </body>
</html>

Bootstrap navbar component
============================
Bootstrap navbar component is useful to creating responsive navigation
header for a website.

A navigation bar is a navigation header that is placed at the top of the page.

A navigation bar cna be extend, collapse depending upon the screen size.

A navigation bar is created by using "navbar navbar-default" class.

"container-fluid" class is used to add hover features.

"navbar-brand" class is used to align elements to left side floating.

ex:
<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
     <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js"></script>
      
  </head>       
  <body >
    <div class="container">
         <nav class="navbar navbar-default">
          <div class="container-fluid">
              <!-- header -->
              <div class="navbar-header">
                    <!-- brand -->
                    <a href="#" class="navbar-brand">GlobalEdx</a>
              </div>
              <ul class="nav navbar-nav">
                  <li><a href="#">HOME</a></li>
                  <li><a href="#">ABOUT</a></li>
                  <li><a href="#">SERVICE</a></li>
                  <li><a href="#">CONTACT</a></li>
              </ul>
          </div>
         </nav>
    </div>
  </body>
</html>

Responsive navigation bar
============================
<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
     <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js"></script>
      
  </head>       
  <body >
    <div class="container">
        <nav class="navbar navbar-default">
          <!-- create a menu icon -->
          <div class="navbar-header">
              <!-- menu bar button -->
              <button class="navbar-toggle" data-toggle="collapse">
                <!-- creat three line -->
                <span class="icon-bar"></span>
                <span class="icon-bar"></span>
                <span class="icon-bar"></span>
              </button>
          </div>
          <!-- create items -->
          <div class="navbar-collapse collapse">
              <ul class="nav navbar-nav">
                  <li class="active"><a href="">HOME</a></li>
                   <li><a href="">ABOUT</a></li>
                    <li><a href="">SERVICES</a></li>
                     <li><a href="">CONTACT</a></li>
              </ul>
          </div>

        </nav>
    </div>
  </body>
</html>

To link the items with toggle button we need to use "data-target" attribute.

data-target attribute always hold id of a element.

ex:
<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
     <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js"></script>
      
  </head>       
  <body >
    <div class="container">
        <nav class="navbar navbar-default">
          <!-- create a menu icon -->
          <div class="navbar-header">
              <!-- menu bar button -->
              <button class="navbar-toggle" data-target="#navid"data-toggle="collapse">
                <!-- creat three line -->
                <span class="icon-bar"></span>
                <span class="icon-bar"></span>
                <span class="icon-bar"></span>
              </button>
          </div>
          <!-- create items -->
          <div class="navbar-collapse collapse" id="navid">
              <ul class="nav navbar-nav">
                  <li class="active"><a href="">HOME</a></li>
                   <li><a href="">ABOUT</a></li>
                    <li><a href="">SERVICES</a></li>
                     <li><a href="">CONTACT</a></li>
              </ul>
          </div>

        </nav>
    </div>
	

  </body>
