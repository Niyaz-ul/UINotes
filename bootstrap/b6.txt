Bootstrap List group
====================
The most basic list group is an unordered list with list items.
ex:

<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
     <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js"></script>
      
  </head>       
  <body>
     Courses
        <ul>
            <li>HTML</li>
            <li>CSS</li>
            <li>JavaScript</li>
        </ul>
  </body>
</html>

To create a basic list group ,use an <ul> element/tag with class "list-group"
and <li> tag/element with class "list-group-item".

ex:


<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
     <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js"></script>
      
  </head>       
  <body>
    <br>
      <div class="container">
        <ul class="list-group">
            <li class="list-group-item">HTML</li>
            <li class="list-group-item">CSS</li>
            <li class="list-group-item">JavaScript</li>
        </ul>
      </div>
  </body>
</html>

ex:

List Groups with Hyperlinks
================================
To create a List Group with hyperlink use <div> instead of <ul> tag and 
use <a> tag instead of <li>.

ex:
<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
     <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js"></script>
      
  </head>       
  <body>
    <br>
      <div class="container">
        <div class="list-group">
            <a href="#" class="list-group-item">First Item</a>
            <a href="#" class="list-group-item">SecondItem</a>
            <a href="#" class="list-group-item">Third Item</a>
        </div>
      </div>
  </body>
</html>

Bootstrap Panels
===================
A panel in bootstrap is a bordered box with some padding around its content.

Panels are created with "panel" class.

The "panel-default" class is used to style the color of the panel.


we can create content inside the panel by using "panel-body" class.

ex:
 
<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
     <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js"></script>
      
  </head>       
  <body>
    <br>
      <div class="container">
            <div class="panel panel-default">
                <div class="panel-body">A Basic Panel</div>
            </div>
      </div>
  </body>
</html>
