Bootstrap Tables
==================
To create table in bootstrap we need to use "table" utility class.

ex:

<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
      <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.16.0/umd/popper.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>

  </head>       
  <body>
      <div class="container">
          <table>
                <tr>
                      <th>ENO</th>
                      <th>ENAME</th>
                      <th>Email</th>
                </tr>
                <tr>
                      <td>111</td>
                      <td>Alan</td>
                      <td>alan@gmail.com</td>
                </tr>  

                <tr>
                      <td>222</td>
                      <td>Jose</td>
                      <td>jose@yahoo.com</td>
                </tr>  

                <tr>
                      <td>333</td>
                      <td>Jessie</td>
                      <td>jessie@hotmail.com</td>
                </tr>  
                <tr>
                      <td>444</td>
                      <td>Karen</td>
                      <td>karen@zoho.com</td>
                </tr>  
          </table>
      </div>
  </body>
</html>

Bootstrap Images
=================
In Bootstrap we can display images in three forms.

1)Rounded Corners   (rounded)

2)Circle	    (rounded-circle) 	

3)Thumbnail	     (img-thumbnail)	


1)Rounded Corners
------------------
The "rounded" class adds rounded circle to an image.

ex:
<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
      <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.16.0/umd/popper.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>

  </head>       
  <body>
      <div class="container">
        <img  src="thumbnail2.jpg" class="rounded"/>
       </div>
  </body>
</html>

Bootstrap Jumbotron
====================
Bootstrap Jumbotron component is generally used to showcase the key content
on a website.

A Jumbotron is displayed as a gray box with rounded corners.

Inside a Jombotron component we can put any valid html tags ,including other
Bootstrap elements /classes.

Use "<div>" element/tag with class "jumbotron" to create a Jumbotron component.

ex:

<!DOCTYPE html>
<html>
  <head>
      <title>MyPage!</title>
      <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.16.0/umd/popper.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>


  </head>       
  <body>
      <div class="container">
          <div class="jumbotron">
              
          </div>
      </div>     
  </body>
</html>

Changing the default Jumbotron background color
============================================
step1:
-----
	check the class for jumbotron using developer option.

step2:
------
	create mystyle.css file for the "jumbotron".

mystyle.css
--------------
.jumbotron
{
	background-color: #0278ef;
}

step3:
---------
	Add external css file to html file using <link> tag.

	ex:
	<!-- add external css file -->
  <link rel="stylesheet" type="text/css" href="mystyle.css">

ex:
index.html
-----------
<!DOCTYPE html>
<html>
