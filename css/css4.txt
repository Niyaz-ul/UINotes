CSS "id" attribute
==================
In CSS, "id" selector can be created by using hash character(#).
syntax
----
#id_name
{
	-
	-list of properties
	-
}

The "id"  specifies a unique id for html elements/tags.

The "id"  is used to point to a specific style declaration in a style sheet.

This "id" is also used by javascript to access and manipulate the element with the
specific id.

problem without id attribute
-----------------------------
<!DOCTYPE html>
<html>
	<head>
		<title></title>
		<style>
				p
				{
						color:blue;
						font-size:20px;
						font-family:arial;
						text-align:center;
				}
		</style>
		
	</head>
	<body >
			<p>This is Paragraph1</p>
			<p>This is Paragraph2</p>
			<p>This is Paragraph3</p>
	</body>
</html>

Here same styles will be applied to all paragraphs.

Solution for above example
---------------------------
<!DOCTYPE html>
<html>
	<head>
		<title></title>
		<style>
				#id1
				{
						color:blue;
						font-size:20px;
						font-family:arial;
						text-align:center;
				}
				#id2
				{
						background-color:red;
						font-size:20px;
				}
		</style>
		
	</head>
	<body >
			<p id="id1">This is Paragraph1</p>
			<p id="id2">This is Paragraph2</p>
			<p>This is Paragraph3</p>
	</body>
</html>


ex:

CSS padding 
==============
The CSS padding properties are used to generate space around elements content.

We havae following CSS properties for specifying padding for each side.

1)padding-top

2)padding-right

3)padding-bottom

4)padding-left

All the above padding properties can have following values.

Note:
-----
	Negative values are not allowed but in margin -ve values are allowed.	
	Margin is page/screen level and padding is a content level.

ex:
<!DOCTYPE html>
<html>
	<head>
		<title></title>
		<style>
				div
				{
					border:2px solid black;
					padding-top:50px;	
					padding-left:50px;
					padding-right:50px;
					padding-bottom:50px;
				}
				
		</style>
		
	</head>
	<body >
			<div>This is Paragraph1</div>
			
	</body>
</html>

syntax
-----

	right - 50px
	bottom - 75px
	left	- 100px


ex:

CSS setting Height and Width
===========================
The "height" and "width" property is used to set the height and width of an element.

The height and width properties do not include padding ,borders and margins.

It set height and width of the area inside the padding ,borders ,margins and other elements.

ex:
<!DOCTYPE html>
<html>
	<head>
		<title></title>
		<style>
				div
				{
					border:1px solid black;
					width: 200px;
					height:50px;
				}
				
		</style>
		
	</head>
	<body >
			<div>This is CSS Divsion Tag</div>
			
	</body>
</html>
