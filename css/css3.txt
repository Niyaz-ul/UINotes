CSS border properties
---------------------
The CSS border properties allows us to specify the style,width and color of 
an element's border.

1)CSS border style
-------------------
The border-style property specifies ,what kind of border to be display.

dotted 		-->	dotted border

dashed		-->	dashed border

solid		-->	solid border

double		-->	double border

groove		-->	groove border

ridge		-->	ridged border

inset		-->	3D inset border

outset		-->	3D outset border

none		-->	no border

hidden		-->	hidden border

ex:1
-----
<!DOCTYPE html>  
 <html>  
    <head>  
		<title>MyPage</title>
	
   </head>  
    <body>  
			<!-- p tag with no border -->
			<p>This is Paragraph Tag </p>
	</body>
</html>
	
Note:
-----
	Bydefault ,For every Element/Tag border will not be displayed.

ex:2
----
<!DOCTYPE html>  
 <html>  
    <head>  
		<title>MyPage</title>
		<style>
			p
			{
				border-style:dotted;
			}
			h1
			{
				border-style:dashed;
			}
			h2{
				border-style:groove;
			}
			div{
				border-style:double;
			}
			h3{
				border-style:solid;
			}
			h4{
				border-style:ridge;
			}
			h5{
				border-style: inset;
			}
			h6{
				border-style:outset;
			}
		</style>
	
   </head>  
    <body>  
			
			<p>This is Paragraph Tag </p>
			<h1>this is Heading 1 tag</h1>
			<h2>This is Heading 2 Tag</h2>
			<div>This is Div Tag</div>
			<h3>This is Heading3 Tag</h3>
			<h4>This is Heading4 Tag</h4>
			<h5>This is Heading5 Tag</h5>
			<h6>This is Heading6 Tag</h6>
			
	</body>
</html>
	
Note:
-----
	The border-style property can have from one to four values i.e for 
	top border, right border ,bottom border and left border.	
	
ex:
<!DOCTYPE html>  
 <html>  
    <head>  
		<title>MyPage</title>
		<style>
			p
			{
				border-style:dotted solid dashed double;
			}
		</style>
   </head>  
    <body>  
			<p>This is Paragraph Tag </p>
	</body>
</html>
	
2)CSS border width
------------------
The border-width property specifies the width of four borders.

The width can be set as a specific size like in px,pt,cm,em and etc.	

We can use one of the predefined value like thin,medium and thick.

ex:
<!DOCTYPE html>  
 <html>  
    <head>  
		<title>MyPage</title>
		<style>
			p
			{
				border-style:solid ;
				border-width:thin;
			}
			h1{
				border-style:solid;
				border-width:medium;
			}
			div{
				border-style:solid;
				border-width:thick;
			}
		</style>
   </head>  
    <body>  
			<p>This is Paragraph Tag </p>
			<h1>This is Heading1 Tag</h1>
			<div>This is div Tag</div>
	</body>
</html>
	
	
ex:
<!DOCTYPE html>  
 <html>  
    <head>  
		<title>MyPage</title>
		<style>
			p
			{
				border-style:solid ;
				border-width:2px;
			}
			h1{
				border-style:solid;
				border-width:.2em;
			}
			div{
				border-style:solid;
				border-width:.1cm;
			}
		</style>
   </head>  
    <body>  
			<p>This is Paragraph Tag </p>
			<h1>This is Heading1 Tag</h1>
			<div>This is div Tag</div>
	</body>
</html>
	
3)CSS border color
--------------------
The border-color property is used to set the color of four borders.

Color can be set by

1)With color name like red.

2)with HEX value like #FFFF00.

3)With RGB value like rgb(255,0,0).

4)With HSL value like hsl(0,100%,50%).	

Note:
----
	Bydefault , all the boders will be displayed with black color.

ex:
<!DOCTYPE html>  
 <html>  
    <head>  
		<title>MyPage</title>
		<style>
			p
			{
				border-style:solid ;
				border-width:2px;
				border-color: violet;
			}
			h1{
				border-style:solid;
				border-width:.2em;
				border-color: #C30F00;
			}
			div{
				border-style:solid;
				border-width:.1cm;
				border-color: rgb(255,0,0);
			}
			h2{
				border-style: solid;
				border:width: 2px;
				border-color: hsl(0,80%,50%);
			}
		</style>
   </head>  
    <body>  
			<p>This is Paragraph Tag </p>
			<h1>This is Heading1 Tag</h1>
			<div>This is div Tag</div>
			<h2>This is heading2 tag</h2>
	</body>
</html>
	
In CSS ,we can set individual sides to an element/tag.

ex:
<!DOCTYPE html>  
 <html>  
    <head>  
		<title>MyPage</title>
		<style>
			p
			{
				border-style:double solid dashed  dotted;
				
			}
			h1{
				border-top-style:double;
				border-right-style:solid;
				border-bottom-style:dashed;
				border-left-style:dotted;	
				
			}
			
		</style>
   </head>  
    <body>  
			<p>This is Paragraph Tag </p>
			<h1>This is Heading1 Tag</h1>
			
	</body>
</html>
	
4)CSS border shorthand property
--------------------------------
It is used to shorthand the property.

It is also possible to declare all the individual properties to one property.

In order to make border shorthand property we need to using following below 
given order.

priorities
-----------
	1)border-width
	2)border-style
	3)border-color	
	
ex:
<!DOCTYPE html>  
 <html>  
    <head>  
		<title>MyPage</title>
		<style>
			p
			{
				border-width:2px;
				border-style:solid;
				border-color:#000000;
				
				
			}
			h1{
				border: 2px solid #000000;	
				
			}
			
		</style>
   </head>  
    <body>  
			<p>This is Paragraph Tag </p>
			<h1>This is Heading1 Tag</h1>
			
	</body>
</html>
	
5)CSS rounded borders
------------------------
The border-radius property is used to add rounded borders to an Element.

ex:
<!DOCTYPE html>  
 <html>  
    <head>  
		<title>MyPage</title>
		<style>
			p
			{
				border:3px solid blue;
				border-radius: 50%;
				
				
			}
			h1{
				border: 2px solid red;	
				border-radius:20px;
				
			}
			
		</style>
   </head>  
    <body>  
			<p>This is Paragraph Tag </p>
			<h1>This is Heading1 Tag</h1>
			
	</body>
</html>

CSS Margins
=============
CSS margin properties are used create space around elements ,outside of 
defined border.


CSS contains following properties to specify the margins for each side of 
an element.

1)margin-top

2)margin-right

3)margin-bottom

4)margin-left


All the above margin properties can have following values . They are 

1)auto 		-	The browser calculates the margin.

2)length	-	It declare margin in px,pt,cm,em and etc.

3)%		- 	It declare margin in percentage(%).

4)inherit	-   	It declare the margin should be inherited from parent element.


ex:
<!DOCTYPE html>  
 <html>  
    <head>  
		<title>MyPage</title>
		<style>
			p
			{
				margin-left: 150px;
				margin-top: 160px;
			}
			
		</style>
   </head>  
    <body>  
			<p>This is Paragraph Tag </p>
			
			
	</body>
</html>

ex:
