			}

			}

				background-size: cover;
				background-attachment: fixed;

CSS pagination
================
If we have website with lots of pages.You may wish to add some
sort of pagination to each page.

ex:
	<  1     2      3    4     5    6    .   .   >

	Previous  1   2   3   4    5   6   .   .    Next

ex:
ex:1
---
<html>
  <head>
	<title>MyPage</title>
	<style>
			.pagination
			{
				display: inline-block;
			}
			.pagination a
			{
				color:black;
				float:left;
				padding: 8px 16px;
				text-decoration: none;
			}
			.pagination a.active
			{
				background-color: red;
				color:white;
			}
			.pagination a:hover:not(.acitve)
			{
				background-color: #dddddd;
			}
	</style>
  </head>
  <body>
			
					<h2>Simple Pagination </h2>
					<div class="pagination">
							<a href="#">&laquo</a>
							<a href="#" class="active">1</a>
							<a href="#">2</a>
							<a href="#">3</a>
							<a href="#">4</a>
							<a href="#">5</a>
							<a href="#">6</a>
							<a href="#">&raquo</a>
					</div>
			
  </body>
</html>
