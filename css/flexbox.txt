CSS Flexbox
=============
Flexbox is a one-dimensional layout method for laying out items in 
rows and columns.

CSS flexbox is a better way to align items into a container.

flexbox = flexible + box.


To create the Flexbox model, you need to first define a "flex container".

ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage</title>
		<style>
				.container
				{
					width:100%;
					height:500px;
					border:2px solid black;
				}
				.item
				{
					background-color: yellow;
					border:2px solid red;
					margin:10px;
					padding: 5px;
				}
		</style>
    </head>
	<body>
		
			<div class="container">
				<div class="item">First box</div>
				<div class="item">Second box</div>
				<div class="item">Third box</div>
				<div class="item">Fourth box</div>
				<div class="item">Fifth box</div>
				<div class="item">sixth box</div>
			</div>

	</body>	
</html>

To create a flexbox we need to use "display:flex" property inside flex container.

Using "display:flex" property all the div's come in inline format.
ex:

Flex properties for Flex container
----------------------------------

1)The flex-direction Property
---------------------------
The flex-direction property defines in which direction the container wants
 to stack the flex items.

flex direction bydefault is row.

ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage</title>
		<style>
				.container
				{
					width:100%;
					height:500px;
					border:2px solid black;
					display: flex;
					/* default direction*/
					flex-direction: row;
				}
				.item
				{
					background-color: yellow;
					border:2px solid red;
					margin:10px;
					padding: 5px;
					width: 200px;
					height:200px;
				}
		</style>
    </head>
	<body>
		
			<div class="container">
				<div class="item">First box</div>
				<div class="item">Second box</div>
				<div class="item">Third box</div>
				<div class="item">Fourth box</div>
				<div class="item">Fifth box</div>
				<div class="item">sixth box</div>
			</div>

	</body>	
</html>

Default value for flex-wrap is "nowrap".

3)The flex-flow property
-----------------------------
If we want to adjust flex-direction and flex-wrap then we need to use "flex-flow" property.
ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage</title>
		<style>
				.container
				{
					width:100%;
					height:500px;
					border:2px solid black;
					display: flex;
					flex-flow: row-reverse wrap;
				}
				.item
				{
					background-color: yellow;
					border:2px solid red;
					margin:10px;
					padding: 5px;
					width: 200px;
					height:200px;
				}
		</style>
    </head>
	<body>
		
			<div class="container">
				<div class="item">First box</div>

				<div class="item">Third box</div>
				<div class="item">Fourth box</div>
				<div class="item">Fifth box</div>

			</div>

	</body>	
</html>

5)The align-content Property
-------------------------------
The align-content property is used to align the flex lines.

T0 stretches the flex lines to take up the remaining space (this is default).
ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage</title>
		<style>
				.container
				{
					width:100%;
					height:500px;
					border:2px solid black;
					display: flex;
					flex-wrap: wrap;
					/* space around */
  					align-content: stretch;
				}
				.item
				{
					background-color: yellow;
					border:2px solid red;
					margin:10px;
					padding: 5px;
					width: 200px;
					height:200px;
					/* item align in center */
					text-align: center;
					line-height: 200px;
				}
		</style>
    </head>
	<body>
		
			<div class="container">
				<div class="item">First box</div>
				<div class="item">Second box</div>
				<div class="item">Third box</div>
				<div class="item">Fourth box</div>
				<div class="item">Fifth box</div>
				<div class="item">sixth box</div>
			</div>

	</body>	
</html>
