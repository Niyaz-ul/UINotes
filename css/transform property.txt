CSS transform
===============
CSS transform property allows you to move ,rotate ,scale ,and skew elements.

CSS tranform property contains following Tranformation method.

translate()
rotate()
scaleX()
scaleY()
scale()
skewX()
skewY()
skew()


translate()
-----------
	The translate() method moves an element from its current 
position to the parameters given by the X-axis and the Y-axis.

ex:
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage</title>
		<style>
			div
			{
				width:300px;
				height:100px;
				background-color:yellow;
				border:1px solid black;
				/* 50 x-axis and 100 y-axis */
				transform: translate(50px,100px);
			}
		</style>		
	</head>
	<body>
			<div>
				This div working form transformation.This div working form transformation.This div working form transformation.This div working form transformation.
			</div>
	</body>	
</html>

-----------

			{
				width:300px;
				height:100px;
				background-color:yellow;
				border:1px solid black;
				transform: rotate(90deg);	
			}
		</style>		
	</head>
	<body>
			<h1>This is Rotate method </h1>
			<div>
				This div working form transformation.This div working form transformation.This div working form transformation.This div working form transformation.
			</div>

</html>
