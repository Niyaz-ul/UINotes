CSS Background
-------------------
The CSS background property is used to set the background in a web page.

The background is defined in the body tag/element which applies to 
our entire web page.

If we want ,we can also set the background to any particular html element.
like <h1>,<i>,<p>,<div> and etc.

The list of following background properties are.

1)background-color

2)background-image

3)background-repeat

4)background-size

5)background-position

6)background-attachment

7)background-shorthand

8)background-blend-mode

9)background-origin

10)background-clip

1)background-color
--------------------
The background-color property specifies the background color of an 
element.

ex:
<!DOCTYPE html>  
 <html>  
    <head>  
		<title>MyPage</title>
		<style>
			body
			{
				background-color:red;
			}
		</style>
   </head>  
    <body>  
	
	
	</body>
</html>
	
In CSS, A color is mostly specifies by 

1)A valid color Name like "red".

2)A HEX value like "#FF0000".

3)A RGB value like rgb(255,0,0).


We can set the background color to any html element/tag.

ex:
<!DOCTYPE html>  
 <html>  
    <head>  
		<title>MyPage</title>
		<style>
			h1{
					background-color:violet;
			}
			div{
					background-color:#FFFF00;
			}
			p{
					background-color:rgb(255,0,0);
			}
		</style>
   </head>  
    <body>  
		<h1>This is heading tag</h1>
		<div>This is division tag</div>
		<p>This is  paragraph tag</p>
	
	</body>
</html>
	
CSS Opacity(Transparency) property
---------------------------------
The Opacity property specifies transparency of an element/tag.

It takes values from 0.0 to 1.0.

3)background-repeat
---------------------
Bydefault background-image property repeats an image horizontally and 
vertically.	
	
The list of background-repeat values are.

1)background-repeat: repeat;

2)background-repeat: repeat-x;

3)backgrond-repeat: repeat-y;

4)background-repeat: no-repeat;

7)background-shorthand
----------------------
This property is used to shorthand the code.

It is possible to specify all the background properties in one single
property using "background-shorthand" property.

ex:
Instead of writing
------------------
background-color: lightgray;
background-image: url("bg2.png");
background-repeat:no-repeat;
background-position:right 0px;	
	
we can use shorthand property
---------------------------
body
{
	background: lightgray url("bg2.png") no-repeat right 0px;
}

8)background-blend-mode
------------------------
The background-blend-node property defines mode of each background layer.

ex:

<!DOCTYPE html>  
 <html>  
    <head>  
		<title>MyPage</title>
		<style>
			body
			{
				background-image: url("bg1.jpg"), url("bg3.png");
				background-repeat: no-repeat,no-repeat;
				background-size: 900px,auto;
				background-position: 0px 0px,450px 0px;
				background-blend-mode: lighten;
				
					
			}
		</style>
   </head>  
    <body>  
		
	</body>
</html>

9)background-origin
--------------------
background-origin property specifies the origin position (background
position area) of a background image.	
	
10)background-clip
------------------
background-clip property specifies how far the background color or image
should be extended within an element.
